## 🚀 Instalando JIU JITSU

Para instalar o JIU JITSU, siga estas etapas:

Windows:

```
npm install
npm start
```

## ☕ Usando JIU JITSU

Para usar JIU JITSU, siga estas etapas:

- Crie uma conta no firebase.
- Faça um link com o projeto.
- Use o firestore database.
